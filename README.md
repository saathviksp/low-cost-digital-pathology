# Low-Cost Digital Pathology

## Goal
Develop an AI-assisted low-cost digital pathology pipeline for blood smear analysis.

## Scope
- Blood cell segmentation
- Malaria infected cell detection
- Morphological feature extraction
- YOLO-based training and inference

## Tech Stack
- Python
- OpenCV
- Roboflow
- YOLOv8
- PyTorch
